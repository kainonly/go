# v1.0.1 - 功能增强与依赖更新

## 🎯 版本概述

这是一个小版本更新，主要包含新功能添加、依赖更新和 CI/CD 工作流优化。此版本专注于增强工具库的实用性和提升项目的稳定性。

## ✨ 新增功能

### UUIDv7 支持
- **help 模块新增 UUIDv7 功能**
  - 支持生成基于时间戳的 UUIDv7 (RFC 9562)
  - 提供从 UUIDv7 中提取时间戳的功能
  - 相比传统 UUID v4，UUIDv7 具有更好的数据库索引性能
  - 适用于需要时间排序的分布式 ID 场景

```go
// 生成 UUIDv7
id := help.UUIDv7()

// 从 UUIDv7 提取时间戳
timestamp, err := help.ExtractUUIDv7Time(id)
```

## 🔧 改进与优化

### 依赖更新
- 更新所有 Go 依赖包至最新稳定版本
- 修复已知安全漏洞
- 提升与新版本 Go 的兼容性
- 优化依赖包的版本管理

### CI/CD 工作流优化
- **统一 Go 版本管理**: 规范化所有工作流中的 Go 版本设置格式
- **简化测试流程**: 优化测试工作流中的 go 模块命令
- **提升可靠性**: 在测试前清理 go 模块缓存，避免缓存污染
- **配置优化**: 移除不必要的缓存配置，简化工作流结构
